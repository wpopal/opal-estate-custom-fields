Create custom fields for Opal Estate Pro plugin. This plugin allows you control and manage fields and used for searchable.

== Description ==

This plugin allows you control and manage fields and used for searchable.

= Features: =

* Create property information metaboxes.
* Manage property information metaboxes searchable.
* Support iconfonts for custom fields.

= Settings: =

* Create property information custom fields: Dashboard > Properties > Custom Fields then create fields. 5 field types are supported (text, textarea, select, checkbox, date).
* Edit a property, visit Information tab to fill data.
* You can manage searchable fields in Dashboard > Properties > Settings > Property > Search

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/opal-estate-packages directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
